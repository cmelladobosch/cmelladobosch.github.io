Projecte Pilot DAW: Natural Language UI

STATUS: PROPOSAL

💡 El Concepte

Transformació radical del desenvolupament web: passar dels Formularis HTML estàtics (inputs rígids) a interfícies basades en Prompts de Llenguatge Natural. L'usuari expressa la intenció, i la IA extreu les dades estructurades (JSON) pel backend.

🛠️ Solució Tecnològica

Local LLM (Llama3/Mistral) Ollama API Structured Outputs (JSON Mode) Pydantic / Zod Validators
Beneficis
  • › UX sense fricció: l'usuari parla com vol.
  • › Accessibilitat nativa per a tothom.
  • › Gestió de dades complexes sense menús infinits.
Reptes (Costos)
  • › Latència: Més lent que un form clàssic.
  • › Al·lucinacions: Risc de dades inventades.
  • › Consum alt de recursos de Hardware.
