Funcionament Bàsic

Aquest simulador executa el cicle clàssic (Fetch-Decode-Execute) de forma visual. Escriu el programa a l'Editor de Codi instrucció per instrucció, i prem Run per executar-lo automàticament o Step per anar pas a pas.

També pots utilitzar l'Assistent d'IA integrat: escriu al camp "Descripció" allò que vols que faci el teu programa, desenfunda i enganxa la teva API Key de Gemini a sota, prement "Generar (IA)". El codi en assemblador s'escriurà tot solet preparat per ser simulat!

Conjunt d'Instruccions (ISA)

  • LOAD [num] - Guarda el número directe [num] a l'Acumulador (ACC).
  • LDA [dir] - Llegeix el número existent a la memòria RAM en la posició [dir] i el guarda a l'ACC.
  • LDAI [dir] - Load Accumulator Indirect: Llegeix el número que hi ha a la RAM a la posició [dir], l'utilitza com a nova adreça (punter), i guarda a l'ACC el valor d'aquella adreça final.
  • STA [dir] - Guarda el valor actual de l'ACC a la memòria RAM en la posició [dir].
  • ADD [dir] - Suma a l'ACC el valor que hi ha a la RAM a la posició [dir].
  • SUB [dir] - Resta a l'ACC el valor que hi ha a la RAM a la posició [dir].
  • JNZ [línia] - Jump if Not Zero: Si l'ACC no és exacte a '0', la CPU salta i continua a la [línia] demanada (comencen per 0).
  • JMP [línia] - Jump: Salt incondicional. La CPU salta a la [línia] demanada i continua independentment de l'ACC.
  • IN - Obre una finestra per demanar que teclegis. Si introdueixes un número, el guardarà directament. Si introdueixes text, en guardarà el codi ASCII de la primera lletra a l'ACC.
  • INS [dir] - Input String: Obre una finestra de text llarg per escriure una paraula o frase. Descompondrà el text en caràcters ASCII i els guardarà ocupant posicions consecutives a la RAM començant des de [dir] fins a posar de límit un 0 al final.
  • OUT - Imprimeix el valor numèric que hi hagi a l'ACC a la Pantalla de Visualització (amb salt de línia).
  • OUTC - Agafa el valor numèric de l'ACC, el converteix al seu caràcter text ASCII i l'imprimeix (sense salt de línia).
  • OUTS [dir] - Output String: Llegeix caràcters consecutius de la RAM començant des de [dir] convertint-los de nou a text, fins a topar amb un 0, mostrant així tota la frase d'un cop.
  • OUTFMT [dir1] [dir2] [dir3] - Imprimeix el text de "X per Y = Z" basat en els valors emmagatzemats a les adreces RAM corresponents.
  • HLT - Halt: Atura i tanca correctament l'execució del programa.

Apunts

La RAM de la CPU pot guardar valors des de la casella 0 fins a la 15.
El codi ignora automàticament qualsevol línia que comenci per "ex:", fent que sigui perfecte per deixar notes/comentaris d'ús.
